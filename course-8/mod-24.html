SAP Web Application Server — монитор сообщений и транзакций
SAP Web Application Server (WAS) — подсистема SAP NetWeaver, обеспечивающая распределение и передачу сообщений и транзакций по узлам многоуровневой клиент-серверной архитектуры технического обеспечения mySAP Business Suite на основе Web-технологий. Функционирование WAS основано на использовании таких Web-стандартов, как XML, SOAP, WSDL, UDDI, XSLT.
Для создания и использования интерфейса в интерактивных режимах обмена сообщениями предусмотрен инструментарий DynPro в виде логики и экранов для презентации бизнес-процессов.
