Клавиатура Брайля
Клавиатура Брайля — устройство для ввода символов текста (символов Брайля) с помощью клавиш с тиснением. Используется людьми с ослабленным или потерянным зрением. Примером компьютера Клавиатура Брайля можеь служить продукт фирмы Syscom (рис. 1). В верхней части компьютера расположены клавиши для набора текста, в нижней части — строка Брайля, которая необходима для чтения текстов и проверки корректности информации, введенной с помощью клавиш. Помимо брайлерской строки, для вывода информации используется сингезатор речи.
