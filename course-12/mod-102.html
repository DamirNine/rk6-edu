LDAP
LDAP (Lightweight Directory Access Protocol) является усовершенствованным вариантом протокола доступа к каталогу DAP (Directory Access Protocol), описанного в спецификации X.500. Протокол DAP предназначен для контроля взаимодействия между системным и пользовательским агентами каталога.
LDAP используется для доступа и обновления информации в каталогах для приложений. Каталоги содержат информацию древовидной структуры. Например, корневая директория может быть "Земной шар", следующий уровень "страны", далее уровни компаний, затем людей, оборудования или документов.
Протокол LDAP является частью стека протоколов TCP/IP
