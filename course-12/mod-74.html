XML Metadata Interchange
XML Metadata Interchange (XMI)— стандарт OMG для обмена метаданными с использованием языка XML. Он ориентирован на обмен метаданными при описании метамоделей с помощью Meta-Object Facility (MOF). Создан прежде всего для обмена между UML и MOF средствами моделирования.
Другой вариант обмена метаданными — использование языка онтологий OWL.
