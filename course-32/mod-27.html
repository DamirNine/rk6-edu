Линии уровня
Построение линий уровня (линий равного уровня) позволяет графически выделить в пространстве исследуемых параметров области с заданными уровнями значений критериев. Примером может служить выделение областей с отрицательными вещественными частями корней системы, т.е. определение областей устойчивости системы, альтернативное D-разбиению. Другим примером служит задача выделения областей, соответствующих предельно допустимым значениям критериев качества.
Линии равного уровня получаются, если трехмерная поверхность пересекается рядом плоскостей, расположенных параллельно друг другу.
Совокупность линий уровня представляет контурный график, который в случае функции двух переменных дает представление об этой функции в виде линий равного уровня.
В пакете MATLAB имеется функция - Изображение линий уровня для трехмерной поверхности
Синтаксис:
contour(Z) contour(x,у,Z) contour(Z,n) contour(x,у,Z,n)
contour(Z,v) contour(x,y,Z,v) contour(..., 'тип_линии')
С = contour(...) [C, h] =contour(...)
Описание:
Команда contour(Z) рисует двумерные линии уровня для массива данных Z, определяющих поверхность в трехмерном пространстве без учета диапазона изменения координат х и у.
Команда contour(x,у,Z), где х и у - векторы, рисует линии уровня для массива данных Z с учетом диапазона изменения координат х и у.
Команды contour(Z,n), contour(x,у,Z,n) рисует n линий уровня для массива данных Z; по умолчанию, n равно 10.
Команды contour(Z, v), contour(x, у, Z, v) рисуют линии уровня для заданных значений, которые указаны в векторе v.
Команда contour(...,'<тип линии>') рисует линии уровня, тип и цвет которых определяются параметром <тип линии > команды plot.
Функция С = contour(...) возвращает массив С описания линий уровней по аналогии с функцией contourc для последующего использования командой clabel.
Функция [С, h] = contour(...) возвращает массив С и вектор-столбец дескрипторов h графических объектов line для каждой линии уровня.
