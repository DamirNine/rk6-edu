Понятие качества систем
Понятие качества системы автоматического управления напрямую связано с целью управления. Количественная оценка качества необходима для параметрического и структурного синтеза систем. Очевидно, именно эта оценка является исследуемой функцией в параметрическом анализе систем. Задачи параметрического и структурного синтеза также состоят в поиске экстремума качественных показателей системы.
Оценка качества — совокупность показателей качества системы автоматического управления которые отражают наиболее важные требования, предъявляемые к системе.
Анализ качества систем управления опирается на следующие важнейшие положения. Во-первых, дается ответ на вопрос, устойчива ли система. Устойчивость систем имеет первостепенное значение при анализе и синтезе систем управления. Инженер-проектировщик должен в первую очередь обеспечить устойчивость системы управления объектом (в общем случае неустойчивым). Устойчивая система - динамическая система (система автоматического регулирования), обладающая ограниченной реакцией на ограниченный входной сигнал. Абсолютная устойчивость - качественный показатель устойчивости системы, т.е. выбор из бинарной пары (да,нет)
Если система устойчива, то рассматривается относительная устойчивость - количественная степень устойчивости системы, а также анализируется поведение системы во времени, т.к. системы объективно являются динамическими. В третьих, для оценки качества систем в детерминированной постановке выбирают типовые воздействия.
Для оценки стохастических систем применяют вероятностные методы, определяя динамическую точность систем автоматического регулирования по значению среднеквадратической ошибки или среднеквадратического отклонения регулируемой переменной.
Показатели качества имеют содержательный смысл, очевидно, только для устойчивых систем, поэтому далее, если не оговорено особо, рассматриваются именно такие системы.
Для класса систем автоматического регулирования выделяют следующие обобщенные показатели качества:
- переходный процесс - процесс, вызванный воздействием на устойчивую систему автоматического регулирования по окончании которого система переходит в новое установившееся состояние;
- статическая точность системы при некоторых типовых воздействиях;
- точность при медленно изменяющихся сигналах.
Выделяют два типа оценок качества регулирования: прямые и косвенные.
