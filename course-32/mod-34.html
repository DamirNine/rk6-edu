Разработка интерфейса для параметрического анализа систем
Графический интерфейс для параметрического анализа систем - инструмент, который призван облегчить организацию исследований, осмысление получены результатов и передачу их для использования во внешних приложениях.
Выделяются следующие группы объектов интерфейса:
- ввод данных о модели и условиях моделирования
- ввод и редактирование параметров, влияние которых следует изучить
- меню для выбора типа параметрических исследований
- ввод дополнительных параметров для организации исследований.
- меню для выбора представления выходной информации
- области графического отображения полученных результатов
- средства передачи информации в другие приложения
- средства управления графическими образами
- средства выхода из программы
- справочные средства, в т.ч. всплывающие подсказки
